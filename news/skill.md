# 경영혁신실 뉴스 클리핑 서비스

## 개요
경영혁신실에서 모니터링해야 할 가상자산 업계 뉴스를 자동으로 수집·분류하여 한눈에 볼 수 있도록 정리하는 웹 기반 뉴스 클리핑 서비스.

- **URL**: [jiminyu.com/news](https://jiminyu.com/news)
- **데이터 소스**: Google News RSS
- **업데이트 주기**: 페이지 접속 시 실시간 수집 (10분 캐시)
- **비용**: 무료 (Vercel + Cloudflare Workers 무료 플랜)

## 뉴스 소스 신뢰도 기준

### Tier 1 — 주요 언론사 (최우선 노출)
동일한 뉴스가 중복될 경우, 아래 언론사의 기사를 우선 노출한다.

| 구분 | 언론사 |
|---|---|
| 국내 종합 | 조선일보, 중앙일보, 동아일보, 한국경제, 매일경제, 서울경제 |
| 국내 경제·금융 | 이데일리, 머니투데이, 파이낸셜뉴스, 한국금융신문 |
| 국내 크립토 전문 | 코인데스크코리아, 블록미디어, 디센터, 토큰포스트 |
| 해외 주요 | Reuters, Bloomberg, WSJ, Financial Times |
| 해외 크립토 전문 | CoinDesk, The Block, Decrypt, CoinTelegraph |

### Tier 2 — 리서치 리포트 (적극 활용)
뉴스 기사와 함께 아래 리서치 기관의 리포트도 포함하여 심층 분석 자료를 제공한다.

| 리서치 기관 | 특징 |
|---|---|
| **4Pillars** | 한국 기반 크립토 리서치, 프로토콜 분석·시장 트렌드 리포트 |
| **Tiger Research** | 아시아 중심 크립토 리서치, 규제·시장 분석 |
| **Xangle** | 온체인 데이터 기반 리서치, 프로젝트 공시·디스클로저 플랫폼 |
| **Messari** | 글로벌 크립토 리서치, 섹터별 분석 리포트 |
| **Delphi Digital** | 심층 프로토콜 분석, 매크로 시장 전망 |
| **a16z Crypto** | 크립토 투자·산업 트렌드 리포트 |

### Tier 3 — 일반 매체
위 Tier 1, 2에 해당하지 않는 매체. 동일 뉴스가 상위 Tier 매체에 있을 경우 노출하지 않는다.

---

## 중복 뉴스 처리 규칙

1. **URL 기반 중복 제거**: 동일 URL의 뉴스는 최초 1건만 표시
2. **동일 내용 우선순위**: 같은 사안을 다룬 기사가 여러 매체에서 수집될 경우, Tier 1 > Tier 2 > Tier 3 순서로 공신력 있는 매체의 기사를 우선 노출
3. **카테고리 간 중복**: 여러 카테고리에서 동일 기사가 잡힐 경우 먼저 로딩된 카테고리에서 1건만 표시

## 기술 구성

| 구성 요소 | 설명 |
|---|---|
| 프론트엔드 | 순수 HTML/CSS/JS (프레임워크 없음) |
| 데이터 수집 | Google News RSS 피드 |
| CORS 프록시 | Cloudflare Workers (`cors-proxy.jiminyu6577.workers.dev`) |
| 호스팅 | Vercel (무료) |
| 캐싱 | 브라우저 메모리 캐시 (10분 TTL) |
