# 📋 위클리 뉴스 클리핑 — 운영 가이드

---

## 1. 실행 방법

### Step 1 — 스킬 실행

Claude Code에서 아래 명령어를 입력합니다.

```
/news-clipping
```

- 기준 기간: **오늘 날짜 기준 1주(7일) 이내** 기사만 수집
- 출력 언어: 한국어
- 자동으로 카테고리별 분류 → 중요도 정렬 → 주요 뉴스 선별까지 수행

### Step 2 — 리포트 저장

생성된 리포트를 아래 경로에 새 파일로 저장합니다.

```
📁 아카이브/2026/YYYY-WNN 주간 뉴스 클리핑 (MM.DD–MM.DD).md
```

**파일명 예시:**
- `2026-W10 주간 뉴스 클리핑 (03.02–03.15).md`
- `2026-W11 주간 뉴스 클리핑 (03.16–03.22).md`

### Step 3 — 홈 업데이트

[[🗺️ 뉴스 클리핑 홈]] 파일에서:
1. **⚡ 최신 리포트** 링크를 새 파일로 교체
2. **📁 아카이브** 표에 새 행 추가

## 3. Slack 공유 (선택)

### 주요 뉴스 실시간 알림

리포트의 **⚡ 이번 주 주요 뉴스** 섹션을 복사하여 Slack 채널에 공유합니다.

### 주간 요약 알림

리포트 전체를 주 1회 Slack에 공유합니다.
→ 향후 n8n 자동화 연동 예정 ([[(hand-written) 기획서]] 참조)

---

## 4. 파일 명명 규칙

| 항목 | 규칙 |
|------|------|
| 연도 | YYYY (예: 2026) |
| 주차 | WNN (ISO 주차, 예: W10) |
| 기간 | MM.DD–MM.DD (시작~종료) |
| 예시 | `2026-W10 주간 뉴스 클리핑 (03.02–03.15).md` |
